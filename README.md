# Restauranttycoon

HTML + PlayCanvas restaurant tycoon prototype.

## Current focus

- Mobile-friendly input
- Fixed 45° camera
- Player movement
- Collisions
- NPC pathfinding
- Interaction system
- Carry and save systems

## Notes

The project is being built as a stable engine foundation first, then restaurant gameplay will be layered on top.

The `Smooth_Male_Casual.fbx` character is intended to replace placeholder player visuals once the import path is finalized.
